# Методология расчёта логистики WB

Источник: [seller.wildberries.ru — Логистика: виды и расчёт стоимости](https://seller.wildberries.ru/instructions/ru/ru/material/logistics-types-and-cost-calculation)
Источник: [seller.wildberries.ru — Фиксация тарифов](https://seller.wildberries.ru/instructions/ru/ru/material/fixation-of-tariffs)

---

## Прямая логистика (доставка покупателю)

Применяется когда `deliveries_count > 0`.

### Формула

```
(delivery_base + delivery_liter × max(0, объём − 1)) × ИЛ + цена × ИРП
```

| Компонент | Описание |
|---|---|
| `delivery_base` | Базовая стоимость первого литра (из `wb_tariffs`, тип `box`). Уже включает `delivery_coef_expr` (коэф. склада) |
| `delivery_liter` | Стоимость каждого дополнительного литра |
| ИЛ | Индекс локализации (из `wb_logistics_indexes.localization_index`) |
| ИРП | Индекс распределения продаж (из `wb_logistics_indexes.irp` ÷ 100) |
| цена | `retail_price_with_discount` из отчёта, либо `avg_price_before_spp` из карточки |

## Обратная логистика (возврат покупателя)

Применяется когда `deliveries_count = 0` и `returns_count > 0`.

**ИЛ и ИРП не участвуют в расчёте** — только базовый тариф по объёму.

### Тарифы по объёму

| Объём | Стоимость |
|---|---|
| 0,001–0,200 л | 23 ₽ |
| 0,201–0,400 л | 26 ₽ |
| 0,401–0,600 л | 29 ₽ |
| 0,601–0,800 л | 30 ₽ |
| 0,801–1,000 л | 32 ₽ |
| > 1 л | 46 ₽ + 14 ₽ × (объём − 1) |

### Ограничение фиксации для возвратов

Обратная логистика фиксируется **только для поставок, отгруженных с 15 мая 2026 года**.
Для более старых поставок тип тарифа всегда «Текущий».

---

## Тип тарифа (Фиксированный / Текущий)

| Источник | Логика |
|---|---|
| `fix_end_date` из отчёта (приоритет) | Если `order_date ≤ fix_end_date` → Фиксированный, иначе → Текущий |
| Fallback по дате поставки | Разница `order_date − supply_date`: < 90 дней → Фиксированный |

### Сроки фиксации по категориям

| Категория | Период |
|---|---|
| Одежда, обувь, спортивная одежда, головные уборы, детская одежда | 90 дней |
| Все прочие категории | **60 дней** |

> **Известное ограничение:** в fallback-логике используется 90 дней для всех категорий,
> т.к. поле `subject` (категория) не передаётся в `wb_weekly_report_rows`.
> Приоритетный путь (`fix_end_date`) определяет тип корректно для всех категорий.

---

## Условия сброса фиксации

Тариф **не** применяется (переходит на текущий), если:
- Поставка доставлена на другой склад или СЦ
- Дата доставки отличается более чем на ±24 часа от планируемой
- Количество товара превысило плановое на ≥100 единиц и >10%
