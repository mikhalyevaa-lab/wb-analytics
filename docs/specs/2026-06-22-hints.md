# Блок E — Подсказки к полям (Hint-компоненты)

## Решения из discovery-interview

- **Иконка**: кружок с `i` (не `?`, не Unicode ℹ️) — единый стиль по всему проекту
- **Триггер**: hover + focus (доступность)
- **Placement**: авто flip top/bottom по viewport; правые колонки в /reports выровнены вправо
- **Страницы**: все три одновременно

## Компонент

`/src/components/ui/hint.tsx` — существовал, обновлён (`?` → `i`)

Props: `children` (React.ReactNode), `size` (sm/md), `align` (left/center/right), `width` (px)

## Реализованные подсказки

### /storage

| Поле | Статус |
|------|--------|
| Всего за период | ✅ добавлена |
| Залежи | ✅ была до блока E |

### /dashboard (kpi-cards + today-cards)

| Поле | Подсказка |
|------|-----------|
| Заказы, шт | Все строки из заказов WB, включая будущие выкупы и отмены |
| Сумма заказов | SUM(finished_price) — включает ещё не выкупленные |
| Реклама, руб | Расходы из API рекламы WB |
| Цена заказа, руб | Бюджет ÷ Заказов |
| Переходов | Клики по рекламным объявлениям |

### /reports (finance-reports)

| Поле | Подсказка |
|------|-----------|
| Продажа | Выручка до вычета расходов |
| К перечислению за товар | После комиссии WB, до логистики/хранения |
| Итого к оплате | Финальная выплата с формулой |
| Прочие удержания/выплаты | Штрафы СЦ, самовыкупы и пр. |
| Стоимость логистики | Доставка + возвраты |
| Корректировка ВВ | Перерасчёт комиссии за прошлые периоды |
| Общая сумма штрафов | Нарушения маркировки, отказы от поставки и пр. |
| Стоимость хранения | Данные еженедельного отчёта (≠ paid_storage API) |
