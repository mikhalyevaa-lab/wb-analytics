## 1. Обзор рынка

Рынок WB-аналитики делится на два лагеря:

**Внешняя аналитика** — анализ рынка, конкурентов, ниш (MPStats, SellerFox).  
**Внутренняя аналитика** — данные из личного кабинета продавца: финансы, склад, заказы (TrueStats, JVO, наш продукт).

Тренд 2025–2026: сервисы сближаются — лидеры добавляют и внешнюю, и внутреннюю аналитику. Доступ к данным конкурентов ограничен WB с сентября 2025, что усиливает ценность внутренней аналитики.

## 2. MPStats

**Сайт:** mpstats.ru  
**Позиционирование:** Рыночный лидер. Комплексная платформа — внешняя + внутренняя аналитика + инструменты продавца.  
**Цена:** от 5 990 ₽/мес, модульная система  
**Маркетплейсы:** WB, Ozon, ЯМ

### Блоки и разделы приложения

| Раздел | Описание |
|--------|---------|
| Внешняя аналитика | WB / Ozon / ЯМ — анализ товаров, категорий, брендов, продавцов |
| Внешняя реклама | Анализ рекламных кампаний конкурентов |
| Кабинет WB | Внутренняя аналитика из личного кабинета |
| Мои карточки | Управление и мониторинг собственных товаров |
| Аудитор SKU 360° | Комплексный аудит каждой карточки |
| Автоответы | Автоматические ответы на отзывы |
| Биддер | Управление ставками в WB Ads |
| SEO | Подбор ключевых слов, оптимизация контента |
| Репрайсер | Автоматическая корректировка цен по конкурентам |
| Плагин (Chrome) | Просмотр аналитики прямо на сайте WB |
| Юнит-калькулятор | Расчёт себестоимости и юнит-экономики |
| Фоторедактор | Обработка фотографий карточек |

### Аналитические вкладки (Внешняя аналитика WB)

- **Товары** — поиск и анализ по артикулам, категориям, ключевым словам
- **Конкуренты** — сравнение продавцов, их ассортимент, динамика продаж
- **Реклама** — анализ рекламных размещений конкурентов
- **Отзывы** — мониторинг отзывов и рейтингов с тональностью
- **Тренды** — динамика спроса по категориям, сезонность
- **Видимость на WB** — позиции в поиске и рекламе
- **Склады и размеры** — распределение остатков по складам и размерной сетке
- **Ценовая сегментация** — анализ ценовых ниш в категории
- **Сравнение периодов** — period-over-period анализ любых метрик

### Визуализация и UX

- Dual-line charts: заказы + выручка на одном графике
- Pie charts: доля рынка по продавцам / брендам
- Таблицы с сортировкой и фильтрацией
- Тепловая карта складских остатков
- Цветовая кодировка трендов (рост/падение)
- Команды (до 4 пользователей в одной подписке)

### Уникальные / killer features

1. **INSIGHT Technology** — собственная технология, показывает реальные продажи даже при скрытых остатках. Ключевое конкурентное преимущество, которое нельзя воспроизвести без доступа к сырым данным WB.
2. **Аудитор SKU 360°** — комплексный аудит каждой карточки с оценкой по 50+ параметрам
3. **AI-анализ конкурентов** — автоматический AI-ответ «что делают лучшие в этой категории»
4. **Поиск товаров-аналогов** — находит похожие товары конкурентов для бенчмаркинга
5. **Chrome плагин** — видеть аналитику прямо на сайте WB без переключения контекста
6. **Биддер + Репрайсер** — автоматизация рекламы и ценообразования в связке

## 3. TrueStats

**Сайт:** truestats.ru  
**Позиционирование:** Внутренняя аналитика с фокусом на финансах и ежедневном контроле прибыли.  
**Цена:** Basic 2 999₽ / Standart 5 999₽ / Pro 9 499₽ / Premium 12 999₽ в месяц  
**Маркетплейсы:** WB, Ozon, ЯМ  
**Trial:** 3 дня бесплатно + 3 месяца данных загружается бесплатно

### Блоки и разделы приложения

| Блок | Что внутри |
|------|-----------|
| **Оцифровка** | 50+ метрик, полная структура прибыли, сравнение периодов, топ товаров по маржинальности |
| **РНП (Рука на пульсе)** | Ежедневный live-мониторинг: динамика заказов/продаж/рекламы, прогноз прибыли и маржи, ДРР, тепловая карта |
| **Аналитика РК** | Тепловая карта кампаний по дням и метрикам, привязка к прибыли, ДРР / CTR / конверсия |
| **Финансы** | ДДС (движения по счетам, структура доходов/расходов) + Платёжный календарь (кассовые разрывы, будущие платежи) |
| **План-факт** | Факт vs план по выручке / продажам / прибыли на уровне артикула, % выполнения, отклонения |
| **Склад** | Остатки на складах и в пути, себестоимость, капитализация, планирование поставок |

### Визуализация и UX

- **Тепловая карта** (РНП и Аналитика РК): матрица дни × метрики — сразу видно паттерны
- Линейные графики динамики по дням / неделям / месяцам
- Прогнозные линии с заливкой области
- Двухуровневые дашборды: сначала агрегаты, затем drill-down по товару
- Таблица «план vs факт» с %-отклонением и цветовой индикацией
- Мобильное приложение с полноценным функционалом
- Telegram-уведомления о ключевых событиях

### Уникальные / killer features

1. **РНП с прогнозом прибыли** — ежедневный живой дашборд: видишь сегодня, прогнозируешь месяц. Самый близкий аналог нашему РНП-блоку.
2. **Платёжный календарь** — визуализация кассовых разрывов и будущих платежей. Уникально на рынке внутренней аналитики.
3. **ДДС (движение денежных средств)** — структура доходов и расходов по счетам, не только WB-выручка.
4. **Мобильное приложение** (iOS + Android + RuStore) — полноценная аналитика на ходу.
5. **3 месяца исторических данных бесплатно** при подключении — снижает барьер входа.
6. **Связка РК → прибыль** — рекламная аналитика привязана к финансовому результату, не к CTR.

## 4. SellerFox

**Сайт:** sellerfox.ru  
**Позиционирование:** Доступная внешняя аналитика, мультимаркетплейс, best value для исследования ниш.  
**Цена:** от 1 950 ₽/мес, есть бесплатный план (500 запросов/день, с рекламой)  
**Маркетплейсы:** WB, Ozon, ЯМ, AliExpress, СберМегаМаркет, KazanExpress

### Блоки и разделы приложения

| Блок | Что внутри |
|------|-----------|
| **Карта трендов** (бесплатно) | Спрос 12 месяцев, прогноз 3 месяца, сезонность — без регистрации |
| **Анализ ниш** | Категории с минимальным числом конкурентов, статистика оборотов |
| **Статистика брендов / продавцов** | Динамика продаж, остатки, выручка |
| **Прогноз продаж** | Прогноз по категориям и брендам |
| **Товары-лидеры** | Максимальный объём и выручка по категориям |
| **AI-инструменты** | Генерация описаний, обработка фото, ответы на отзывы |

> ⚠️ **SellerFox — только внешняя аналитика.** Нет кабинета продавца, финансов, склада, заказов.

### Визуализация и UX

- Карта трендов как центральный элемент (первое, что видит пользователь)
- Минималистичный интерфейс, ориентированный на поиск ниш
- Таблицы с сортировкой по выручке / продажам / конкуренции
- Бесплатный режим с ограничениями (мотивирует на апгрейд)
- Chrome плагин

### Уникальные / killer features

1. **Бесплатная карта трендов** — спрос и прогноз без регистрации. Мощный инструмент привлечения.
2. **Самый дешёвый платный тариф** на рынке (1 950 ₽) при годовой глубине аналитики.
3. **AI-генерация контента** — описания товаров, ответы на отзывы прямо в сервисе.
4. **6 маркетплейсов** в одном интерфейсе — включая нишевые (KazanExpress, AliExpress).
5. **Бесплатный план с 500 запросами/день** — функциональный free tier, а не просто демо.

## 5. JVO (AI CFO)

**Позиционирование:** AI-driven платформа ежедневных заданий для роста продаж на WB. Превращает аналитику в конкретные действия.  
**Цена:** по запросу (корпоративный продукт)  
**Маркетплейсы:** WB (основной)

> Примечание: В ROADMAP_V2.md этот сервис обозначен как «AI CFO» и упоминается как конкурентный ориентир. По описанию функционала соответствует платформе JVO.

### Блоки и разделы приложения

| Блок | Что внутри |
|------|-----------|
| **Ежедневные задания** | AI генерирует персонализированный список задач для роста |
| **Скоринг карточек** | Оценка каждой карточки по влиянию на ранжирование и конверсию |
| **Подсортировщик** | Расчёт отгрузок с учётом сезонного спроса по регионам |
| **Финансы** | Рекомендации по контролю маржинальной прибыли и рентабельности |
| **Региональные отгрузки** | Прогноз спроса по округам и складам для каждого артикула |
| **SEO-оптимизация** | Подбор ключевых слов и оптимизация контента карточек |
| **Аналитика конкурентов** | Детальный разбор действий конкурентов через Telegram-бот |

### Визуализация и UX

- Дашборд «точек роста» — автоматически собирается каждый день
- Ежедневный список задач с приоритетами и прогнозируемым эффектом
- Уведомления через Telegram-бот
- Акцент на actionability: не «что происходит», а «что делать»

### Уникальные / killer features

1. **Ежедневные AI-задания** — не просто аналитика, а конкретные действия с прогнозируемым результатом. Geймификация работы продавца.
2. **Скоринг карточек** — количественная оценка каждой карточки, что именно мешает продавать.
3. **Подсортировщик** — самый точный расчёт поставок с учётом сезонности и региональных складов.
4. **AI как CFO** — финансовые рекомендации на основе данных, а не просто отчёты.

### Есть у нас, нет у конкурентов (наши преимущества)

- **Profit Waterfall** — визуализация структуры прибыли в виде водопада. TrueStats показывает структуру, но не waterfall-визуально.
- **Детальная аналитика хранения** — расхождения ×17-23 задокументированы и исправляются. Конкуренты не делают такую сверку.
- **Кастомная сверка с отчётами WB** — наш инструмент сверяет данные API с еженедельными отчётами. Нигде нет.
- **Глубокая интеграция с данными** — 79 054 строк продаж, 236 отчётов. Больше глубина данных = точнее аналитика.

### Sellmonitor

**Сайт:** sellmonitor.com  
**Позиционирование:** Профессиональная платформа №1 по качеству данных (Adindex). Внешняя + внутренняя аналитика + полная автоматизация продаж. Работает с 2019 года.  
**Цена:** от 19 000 ₽/мес (демо по запросу)  
**Маркетплейсы:** WB, Ozon, ЯМ + 20+ площадок

#### Блоки и разделы

| Блок | Что внутри |
|------|-----------|
| **Товары** | История цен, остатков, выручки, продаж; позиции в поиске; упущенная выручка; история доступности; история изменений контента карточки |
| **Категории / магазины / бренды** | Выручка, продажи, средняя цена; история и сезонность; распределение по ценовым диапазонам; оценка монополизации ниши |
| **Поисковые запросы** | История популярности и сезонности запроса; конкуренты по запросу; запросы по категориям; тренды |
| **Внутренняя аналитика** | Финансы: выручка, расходы, прибыль по магазину и товару; история расходов; ABC-анализ прибыли; история заказов |
| **Планирование поставок** | Какие товары требуют поставки; расчёт количества и суммы; учёт регионов и индекса локализации; детальный расчёт к каждому товару |
| **Управление рекламой (Биддер)** | Каждые несколько минут корректирует ставку; тепловая карта «когда реклама работает лучше всего»; кластеры позиций/просмотров/кликов/расходов; автопополнение бюджета и лимит/день |
| **Управление ценами (Репрайсер)** | Мониторинг каждую минуту; фиксированная цена (защита от акций); история изменений с причинами; расписание по времени суток / остаткам / ценам конкурентов |

1. **ДРР с раздельным CPC и CPO** — ЛК Ozon их смешивает. Yoolip разделяет. Ключевой pain для рекламодателей.
2. **Смарт-правила** — автоматическое обнаружение проблемных кампаний (расход без продаж, перерасход). Не надо ежедневно проверять вручную.
3. **Chrome Extension с маржой в карточке** — парсит данные и показывает маржу прямо на странице товара. Не нужно копировать в калькулятор.
4. **Unit-экономика с FBO/FBS/rFBS** — все схемы в одной таблице; учёт карго и курса ЦБ.
5. **«Аналитика без Ozon Premium»** — конкурирует с Ozon, а не с аналитическими сервисами. Уникальное позиционирование.
6. **Ценообразование по обороту** — Аналитика от 0 ₽/мес. Снижает порог входа до минимума.

#### Дорожная карта Yoolip (анонсировано)

- AI-аналитика рекламных кампаний (май 2026)
- Фактическая юнит-экономика (май 2026)
- Планирование поставок (июнь 2026)
- **Wildberries** (анонсировано — прямой будущий конкурент)

> ⚠️ Стратегически важно: Yoolip выходит на WB. С учётом их фокуса на рекламной аналитике и Chrome Extension — прямая угроза в нише РК + юнит-экономики.

#### Инсайты для нашего продукта

1. **Разделение CPC/CPO в аналитике РК** — мы тоже должны явно разделять типы рекламных расходов.
2. **Смарт-правила** — автоматические алерты о проблемах без ручного мониторинга. Добавить в v5.0.
3. **Chrome Extension с маржой в карточке** — важный acquisition tool; пользователи открывают карточку конкурента и видят маржу прямо там.
4. **Позиционирование против платформы** (Ozon Premium) — сильный фрейм: «мы даём то, за что платформа берёт деньги». Можно использовать аналогично vs. WB Аналитика.

## 14. Выводы

**MPStats** — недостижимый уровень для нас в external analytics, но внутренняя аналитика сопоставима. Их killer feature (INSIGHT tech) невоспроизводима без партнёрства с WB.

**TrueStats** — ближайший прямой конкурент по набору функций. Их РНП, ДДС, Платёжный календарь и мобильное приложение — чёткие ориентиры для v5.0. Ценник сопоставим.

**SellerFox** — не конкурент напрямую (только внешняя аналитика), но показывает ценность: бесплатная карта трендов как acquisition инструмент.

**JVO** — интересная модель «аналитика → задачи». AI-рекомендации на основе данных — направление для долгосрочной дифференциации.

**Наши преимущества:** глубина сверки данных, Profit Waterfall, кастомные алгоритмы расчёта, детализация хранения. Нужно превращать это в пользовательский нарратив.

**Стратегический приоритет v5.0:** замкнуть финансовый контур (ДДС + Платёжный календарь), добавить Telegram-нотификации, улучшить мобильную доступность.
